# REF-NEAT-2002 — Stanley & Miikkulainen (Evolutionary Computation, 2002)

Mécanisme cœur : la **spéciation par distance topologique** protège les structures immatures
le temps qu'elles mûrissent ; `add_node` = split d'arête (innovation neutre tracée par
marqueurs historiques).

Pont AGAGI : EDR-060 redécouvre « la spéciation protège la nouveauté » (presque mot pour mot) ;
EDR-064 redécouvre que `add_node` non protégé = bloat neutre. Notre évolution n'a **pas** de
vraie spéciation par distance (juste bins de taille) → adopter NEAT-python ou son opérateur
de spéciation comme baseline plutôt que le réimplémenter.
