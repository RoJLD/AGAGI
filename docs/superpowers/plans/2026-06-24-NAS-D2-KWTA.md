# D2 KWTA modéré (sparsité imposée) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Imposer une sparsité modérée sur les nœuds cachés du connectome (KWTA) et mesurer si elle est gratuite (compétence préservée) ou coûteuse — le pivot après la réfutation de D1.

**Architecture:** (1) Mécanisme cœur : masque k-winners-take-all par agent sur les cachés dans `MambaBatchModel.forward`, gaté par l'attribut de classe `KWTA_KEEP_FRAC` (1.0=off), propagé depuis `WorldConfig.kwta_keep_frac`. (2) Mesure : généraliser l'outil X2 existant pour balayer un knob de config arbitraire, réutilisé pour `kwta_keep_frac`.

**Tech Stack:** Python 3, NumPy, pytest.

**Spec:** [`../specs/2026-06-24-NAS-D2-KWTA-design.md`](../specs/2026-06-24-NAS-D2-KWTA-design.md)

## Global Constraints

- **Non-régression bit-exacte** : `KWTA_KEEP_FRAC=1.0` et `kwta_keep_frac=1.0` par défaut ⇒ garde `if keep < 1.0` ⇒ aucun masque ⇒ prod inchangée.
- **KWTA sur les CACHÉS uniquement** : positions `mappings[i][I_i : N_i-O_i]`. Entrées (`[0:I_i]`) et sorties (`[N_i-O_i:N_i]`) JAMAIS masquées (politique préservée).
- **Attribut de classe, pas kwarg `__init__`** : pour ne pas casser le seam `batch_model_cls` (BaselineBatchModel S2). Pattern identique à `ABLATE_*`/`METABOLIC_ACTIVE_EPS`.
- **Rétro-compat outil** : les défauts (`param="metabolic_cost_coef"`, `baseline=0.0`) laissent les runs D1 identiques.
- **Git** : commits PATH-SCOPED (`git add <paths>`), jamais `-A`/`.`. Trailer `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.

def _forward_capture(keep_frac):
    np.random.seed(0)
    a = MambaAgent()
    np.random.seed(123)
    m = MambaBatchModel([a])
    MambaBatchModel.KWTA_KEEP_FRAC = keep_frac
    I = a.genome.num_inputs
    np.random.seed(123)
    m.forward(np.ones((1, I), dtype=np.float32))
    MambaBatchModel.KWTA_KEEP_FRAC = 1.0  # reset global
    return m.H_prev_batch.copy(), m.mappings[0], a.genome


def test_kwta_sparsifies_hidden_preserves_io():
    H_off, map_idx, g = _forward_capture(1.0)
    H_on, _, _ = _forward_capture(0.5)
    I, O, N = g.num_inputs, g.num_outputs, g.num_nodes
    # Entrées et sorties INCHANGÉES par KWTA
    np.testing.assert_array_equal(H_off[0, map_idx[:I]], H_on[0, map_idx[:I]])
    np.testing.assert_array_equal(H_off[0, map_idx[N - O:N]], H_on[0, map_idx[N - O:N]])
    # Cachés : KWTA 0.5 a STRICTEMENT plus de zéros que KWTA off
    hid_off = H_off[0, map_idx[I:N - O]]
    hid_on = H_on[0, map_idx[I:N - O]]
    assert np.count_nonzero(hid_on) < np.count_nonzero(hid_off)
    assert np.count_nonzero(hid_on) >= 1   # jamais totalement éteint

        # Gated (1.0 = off). Appliqué avant l'extraction des sorties ET avant H_prev_batch -> le
        # compteur D1 (activation_cost_batch) et l'état récurrent reflètent la sparsité.
        keep = MambaBatchModel.KWTA_KEEP_FRAC
        if keep < 1.0:
            for i in range(self.B):
                N_i = self.agents[i].genome.num_nodes
                I_i = self.agents[i].genome.num_inputs
                O_i = self.agents[i].genome.num_outputs
                hidden_pos = self.mappings[i][I_i:N_i - O_i]
                n_hidden = len(hidden_pos)
                if n_hidden == 0:
                    continue
                n_keep = max(1, int(np.ceil(keep * n_hidden)))
                if n_keep >= n_hidden:
                    continue
                h_vals = H[i, hidden_pos]
                top = np.argsort(-np.abs(h_vals))[:n_keep]
                masked = np.zeros(n_hidden, dtype=h_vals.dtype)
                masked[top] = h_vals[top]
                H[i, hidden_pos] = masked
```

def run_sweep(seeds, coefs, eras: int = 15, num_agents: int = 30, max_ticks: int = 400,
              run_era_fn: Optional[Callable] = None, param: str = "metabolic_cost_coef",
              baseline: float = 0.0) -> Dict:
    """Sweep apparié d'un knob de config arbitraire : par seed, chaque valeur -> run_lineage. Ratios vs baseline -> verdict."""
    coefs = list(coefs)
    if baseline not in coefs:
        coefs = [baseline] + coefs
    per_lineage = []
    by_seed: Dict[int, Dict[float, Dict]] = {}
    for seed in seeds:
        by_seed[seed] = {}
        for coef in coefs:
            r = run_lineage(seed, coef, eras, num_agents, max_ticks, run_era_fn, param=param)
            by_seed[seed][coef] = r
            per_lineage.append(r)
    per_coef = []
    for coef in coefs:
        if coef == baseline:
            continue
        eff_ratios, surv_ratios = [], []
        for seed in seeds:
            base, cur = by_seed[seed][baseline], by_seed[seed][coef]
            eff_ratios.append(cur["efficiency"] / max(base["efficiency"], 1e-6))
            surv_ratios.append(cur["survival"] / max(base["survival"], 1e-6))
        per_coef.append({"coef": coef, "eff_ratios": eff_ratios, "surv_ratios": surv_ratios})
    verdict = compute_sweep_verdict(per_coef)
    return {**verdict, "per_lineage": per_lineage,
            "config": {"seeds": [int(s) for s in seeds], "coefs": coefs, "eras": eras,
                       "num_agents": num_agents, "max_ticks": max_ticks,
                       "param": param, "baseline": baseline}}
```
